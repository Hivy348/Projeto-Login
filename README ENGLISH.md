This project was created with the purpose of being an initial portfolio product. It involves the development of a login and registration page with a floral theme, using HTML and CSS, with a small JavaScript application generated with the help of Artificial Intelligence for basic interactions, and Canva for design creation.
The goal is to present a visually pleasant, cheerful, responsive and functional interface for user authentication in web applications.

This page includes Login and Registration sections with the following elements:

🌼 Floral Background

🔐 Login
- A flower centered above the title
- Username and Password fields
- Login button
- Link to the registration page
- Social media(Facebook, X, and Gmail)
  
📝 Registration

- A flower centered above the title
- Name 
- Email
- Password
- Register button
  
✍️ Authorship

Project developed as a practice exercise in HTML and CSS.
All JavaScript logic was created with the assistance of ChatGPT as a learning support tool.
