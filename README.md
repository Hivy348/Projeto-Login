# Projeto-Login e Cadastro
Este projeto foi feito com o objetivo de ser um produto inicial de projeto para portfólio. Consiste no desenvolvimento de uma página de login e cadastro com temática de flor utilizando em maior foco HTML e CSS,com uma pequena aplicação em Javascript feita por Inteligência Artificial para interações básicas e uso do Canva para criação do design.
O objetivo é apresentar uma interface visualmente agradável,alegre,responsiva e funcional para a autenticação de usuários em aplicações web.

Nesta página contém login e cadastro com:

🌼Fundo florido

🔐Login
- Uma flor centralizada em cima da palavra
- Usuário e Senha
- Entrar
- Link para a página de cadastro
- Redes Sociais(Facebook,X e Gmail)

📝Cadastro:
- Uma flor centralizada em cima da palavra
- Nome
- E-mail
- Senha
- Cadastrar
- Link de retorno a página de login quando já tem conta

Autoria
Projeto desenvolvido como prática de HTML e CSS e o uso do Canva.
Toda a lógica em Javascript foi feita com o auxílio do ChatGPT como ferramenta de apoio e aprendizado.
